# Hospital Patient Records Dashboard

An interactive Excel dashboard built to analyze hospital patient data, track key performance indicators (KPIs), and generate actionable insights for hospital operations, quality, and finance teams.

## 🎯 Project Overview

This project demonstrates end-to-end data analytics skills using a synthetic hospital dataset (1,000 patient records). The dashboard helps hospital leadership quickly understand:

- Patient volume and demographics
- Operational efficiency (Length of Stay)
- Financial performance (Average Cost)
- Quality metrics (Readmission Rate & Patient Satisfaction)
- Condition-level trends

Built as a **professional-grade Excel solution** suitable for real hospital reporting.

## 📊 Key Features

- **Interactive Dashboard** with slicers for dynamic filtering (Gender, Age Group, Condition, Outcome)
- **Executive KPI Cards**: Total Patients, Avg Length of Stay, Avg Cost, Avg Satisfaction
- **Visualizations**:
  - Top conditions by volume and cost
  - Length of Stay by age group
  - Readmission analysis
  - Cost vs Satisfaction scatter plot
  - Gender and outcome breakdowns
- **Insights Sheet** with key findings and recommendations
- **Notes & Methodology Sheet** for full documentation and reproducibility

## 🗂️ Dataset

- **Source**: [Hospital Dataset for Practice](https://www.kaggle.com/datasets/blueblushed/hospital-dataset-for-practice) (Kaggle)
- **Records**: 1,000 synthetic patient records
- **Columns**: Patient_ID, Age, Gender, Condition, Procedure, Cost, Length_of_Stay, Readmission, Outcome, Satisfaction
- **License**: Apache 2.0

## 🛠️ Tools & Skills Demonstrated

- Microsoft Excel (Tables, PivotTables, Slicers, Charts, Conditional Formatting)
- Data cleaning and preparation
- Calculated columns (Age_Group, Cost_Bucket, LOS_Category, Readmission_Flag)
- Dashboard design best practices
- Deriving actionable business insights from raw data
